DeepSpeed 是微软开源的**大模型训练加速与显存优化框架**，核心目标是解决 “大模型显存不够用、训练效率低” 的问题。
- **突破显存瓶颈**：它能把训练中占用显存最大的**优化器状态（Optimizer States）和梯度（Gradients）** 从 GPU 显存 “卸载” 到 CPU 内存中，只保留模型参数在 GPU 上。
- **性价比提升**：你的 4090 只有 24G 显存，用 LoRA 微调 7B 模型可能刚好，但如果想**增大 Batch Size、微调更大的模型（如 13B）、或者开启全量微调**，原生 PyTorch 会直接显存溢出（OOM），而 DeepSpeed 能通过 “借” CPU 内存让你跑起来
#### 💩关键概念
- **ZeRO Stage 0**：不做显存拆分，仅用基础功能。
- **ZeRO Stage 2**：拆分优化器状态和梯度，配合 Offload 到 CPU，是**单卡性价比最高的选择**。
- **ZeRO Stage 3**：拆分模型参数、梯度、优化器（甚至能把模型层也卸载到 CPU），显存占用最低，但单卡下速度损失比 Stage 2 大。
