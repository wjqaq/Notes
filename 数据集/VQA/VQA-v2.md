#### ⌚背景
VQA v1 [VQA(Visual Question Answering)](VQA(Visual%20Question%20Answering).md)数据集虽推动了领域发展，但存在致命的**语言先验 / 偏差缺陷**
##### 1. **强语言统计偏置**：
原数据集中，问题与答案之间存在极强的统计关联，模型无需看图像，仅靠问题文本就能获得极高准确率。例如：
- 对以`Do you see a...`开头的问题，盲答`yes`就能达到 87% 的准确率；
- 对`What sport is...`类问题，仅答`tennis`就能命中 41% 的正确答案；
- 对`How many...`类问题，仅答`2`就能命中 39% 的正确答案。
##### 2. **模型能力被严重高估**：
当时的 SOTA 模型在 v1 上的高分，本质是拟合了语言偏置，而非真正实现了跨模态视觉 - 语言理解，甚至纯语言模型就能达到近 50% 的准确率，完全违背了 VQA 任务 “视觉理解为核心” 的初衷。
##### 3. **现有平衡方案的局限性**：
此前的去偏差工作仅针对二元 yes/no 问题、人工抽象场景，无法适配真实图像场景，也无法覆盖全类型的问答任务。
#### 🧬结构
VQA-v2 的核心设计：对原数据集中的每一个`(图像I, 问题Q, 答案A)`三元组，额外采集一个**互补图像 I'**，满足两个核心条件：
1. 语义相似性：I' 与 I 在视觉语义空间中高度相似；
2. 答案互斥性：同一个问题 Q，在 I' 上的标准答案 A' 与原答案 A 完全不同。

| 维度         | 详细参数                                                                                                                      |
| ---------- | ------------------------------------------------------------------------------------------------------------------------- |
| 总图像 - 问题对  | 超 110 万对（VQA v1 约 55 万对）                                                                                                  |
| 覆盖 COCO 图像 | 约 20 万张，与 VQA v1 完全一致                                                                                                     |
| 配套答案总量     | 约 1300 万条，每条问题均配套 10 个人工标注答案                                                                                              |
| 训练集        | 443K 图像 - 问题对                                                                                                             |
| 验证集        | 214K 图像 - 问题对                                                                                                             |
| 测试集        | 453K 图像 - 问题对，额外补充 18K 对用于异常检测；沿用 v1 的 4 个子拆分：test-dev（调试用）、test-standard（公开榜单）、test-challenge（竞赛评定）、test-reserve（防过拟合校验） |
同时，VQA-v2 也沿用了 v1 的两大任务范式：**开放式问答（Open-Ended）** 和 **选择题问答（Multiple-Choice）**，评估指标、文本预处理规范与 v1 完全一致。
