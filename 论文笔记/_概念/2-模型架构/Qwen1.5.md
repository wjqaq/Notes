# Qwen1.5

## 定义
阿里通义千问团队发布的 Qwen 系列第一代到第二代之间的过渡版本，包含 0.5B-110B 多种规模以及 MoE 模型。

## 核心要点
1. 模型规模: 0.5B / 1.8B / 4B / 7B / 14B / 32B / 72B / 110B + MoE-A2.7B
2. 作为 Qwen2 的直接前身和主要对比基线
3. 上下文窗口: 4K tokens（Qwen2 扩展至 32K）
4. 预训练数据: 约 3T tokens（Qwen2 扩展至 7T）
