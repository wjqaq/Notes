# π0

物理智能 (Physical Intelligence) 提出的 VLA 模型，使用 Flow Matching 进行连续动作生成，采用 2-expert MoT 架构（VLM + 动作专家）。

- 论文: [π0: A Vision-Language-Action Flow Model for General Robot Control](https://arxiv.org/abs/2410.24164) (2024)
- 架构: MoT routing，连续动作头（非离散 token）
- 关系: [[UAM]] 的 2-expert 基线
