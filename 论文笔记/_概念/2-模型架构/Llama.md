# Llama

## 定义
Meta AI 发布的一系列开源自回归语言模型家族，是当前开源 LLM 生态的标杆之一。

## 核心要点
1. Llama 1 (2023): 7B-65B，开源研究社区重要里程碑
2. Llama 2 (2023): 增加 RLHF 对齐的 chat 版本
3. Llama 3 (2024): 8B/70B，GPT-4 级别的开源模型
4. Llama 4 (2025): 引入 MoE 架构
