# NVIDIA A100 GPU

NVIDIA A100 是基于 Ampere 架构的数据中心 GPU，配备 80GB HBM2e 显存。广泛用于深度学习训练和推理。

## 特点

- 80GB HBM2e 显存
- 支持 TF32、FP16、BF16、INT8 等精度
- Multi-Instance GPU (MIG) 支持

Entity-Rubrics 论文使用单张 A100 80GB 完成所有开源模型的推理，总计约 150 GPU 小时。
