# 交叉注意力

## 定义
在多模态模型中，一种模态的 token（如文本）通过注意力机制从另一种模态的 token（如图像）中聚合信息的机制。在大视觉语言模型中，交叉注意力的强度决定了[[视觉定位]]的质量。

## 核心要点
1. **在 LVLM 中**：通常视觉 token 被拼接到文本 token 序列前方，通过 causal self-attention 实现跨模态交互
2. **长程衰减**：由于 [[RoPE]] 位置编码，远端文本 token 与图像 token 的距离增大，交叉注意力自然减弱
3. **替代架构**：[[Q-Former]] 使用专用的 query token 通过交叉注意力提取视觉特征

## 代表工作
- [[InstructBLIP]]: 使用指令感知 [[Q-Former]] 实现条件式交叉注意力
- [[FLB]]: 通过复用首 token logit 间接补偿交叉注意力衰减
- [[Concentric Causal Attention]]: 通过修改注意力掩码保持图像 token 对远端文本的可达性
