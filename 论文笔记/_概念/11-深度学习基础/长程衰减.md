# 长程衰减

## 定义
在 [[大视觉语言模型]] 的自回归生成过程中，随着序列增长，视觉 token 与远端文本 token 之间的[[交叉注意力]]逐渐减弱，导致[[语言先验]]取代视觉定位占据主导的现象。

## 核心要点
1. **根因**：[[RoPE|Rotary Position Embedding]] 将图像 token 置于序列开头，随着生成推进，图像与当前 token 的相对位置距离增大，跨模态注意力衰减
2. **后果**：后期 token 更容易产生[[物体幻觉]]，因为模型越来越依赖语言统计规律而非视觉证据
3. **现有方法的不足**：[[对比解码]] 在每步局部对比 logit，但无法保持全局视觉定位
4. **FLB 的解决思路**：复用距离最近（第一步）的视觉信号 $l_0$，通过[[时变权重函数]]持续注入

## 数学直觉

在 [[RoPE]] 中，位置 $m$ 和 $n$ 之间的注意力权重为：

$$
a_{mn} \propto \text{RoPE}(q_m, k_n) \propto \cos((m-n)\theta)
$$

当 $|m-n|$ 增大时，高频分量产生相位差，注意力自然减弱。

## 代表工作
- [[FLB]]: 通过复用首 token logit 对抗长程衰减
- [[Concentric Causal Attention]]: 通过修改注意力机制（使图像 token 保持与所有文本 token 的距离）来缓解
