# Language Prior

## 定义
语言先验指 LVLM 在生成文本时倾向于依赖语言学上下文中的统计共现模式，而非严格基于视觉证据做出预测。

## 核心要点
1. 是 LVLM 幻觉的重要来源：物体共现模式（如"桌子"和"椅子"频繁同现）导致模型在视觉证据不足时错误推断
2. 在 POPE 的 Adversarial 和 Popular 负采样设置下尤为明显
3. 与[[视觉接地]]形成对立：语言先验主导的 token 缺乏视觉证据支撑
4. 对比解码方法通过构造参考来暴露和抵消语言先验的影响

## 代表工作
- [[SIRA]]: 通过共享前缀 + 后期视觉掩码构造内部语言先验参考，实现 token 级先验抵消
- [[VCD]]: 通过外部输入扰动构造对比参考
