# 边界框检测

## 定义
预测目标物体在图像中的矩形包围框位置和大小的计算机视觉任务。

## 数学形式
$$
\mathcal{L}_{\text{det}} = \mathcal{L}_{\text{smooth-L1}}(\hat{y}_{\text{bbox}}, y_{\text{bbox}}) + \mathcal{L}_{\text{GIoU}}(\hat{y}_{\text{bbox}}, y_{\text{bbox}})
$$

## 核心要点
1. 输出格式：通常为 (x, y, w, h) 或 (x1, y1, x2, y2)
2. 常用损失：Smooth L1 + GIoU (广义 IoU)，GIoU 考虑了非重叠情况
3. BTL (Bounding-box as Language Token)：将边界框坐标编码为文本 token 训练——但可能损害 MLLM 推理能力
4. 在 MLLM 接地中：边界框提供目标级视觉证据
