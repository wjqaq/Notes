# 直接视觉定位效应

## 定义
[[FLB]] 的第一重机制：首 token logit 中蕴含最强的视觉信息（ground truth token logit 显著高于 hallucination token logit），通过复用该 logit 在所有解码步中持续强化视觉定位，直接对抗[[长程衰减]]。

## 核心要点
1. **定量证据**：首 token 中 ground truth 词的 logit 显著高于 hallucination 词的 logit
2. **作用方式**：通过[[时变权重函数]]将这一 logit 优势反复施加到每个解码步
3. **消融验证**：仅保留该效应（屏蔽 "The" 相关 logit）可在 baseline 上改善 CHAIR（11.9 → 9.2）
