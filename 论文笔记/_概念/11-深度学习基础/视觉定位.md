# 视觉定位

## 定义
模型将生成的文本内容与视觉输入中的实际物体/区域建立对应关系的能力。在多模态模型中，视觉定位强度决定了模型抗拒[[语言先验]]、避免[[物体幻觉]]的能力。

## 核心要点
1. **衡量方式**：可通过 ground truth token 与 hallucination token 的 logit 差值间接度量
2. **衰减**：受 [[RoPE]] 位置编码影响，视觉定位随序列长度增加而减弱（[[长程衰减]]）
3. **FLB 的发现**：首 token 的视觉定位最强，因为此时离视觉 token 最近

## 代表工作
- [[FLB]]: 提出直接视觉定位效应和隐式视觉引用效应两种视觉定位增强机制
- [[VCD]]: 通过对比解码局部增强视觉定位
