# 双流假说（Two-Stream Hypothesis）

## 定义

灵长类视觉皮层的双流组织理论 [Goodale & Milner, 1992; Ungerleider, 1982]：

- **腹侧通路（Ventral Stream / "What" pathway）**: 从 V1 → V4 → IT，处理物体识别、语义理解
- **背侧通路（Dorsal Stream / "Where/How" pathway）**: 从 V1 → MT → 后顶叶皮层，处理空间定位、视觉运动控制

两个通路保持功能上独立的表征，使得运动技能的精炼不会干扰语义识别。
