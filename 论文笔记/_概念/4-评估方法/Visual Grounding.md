# Visual Grounding

## 定义
视觉接地指 LVLM 生成的文本内容能否在输入图像中找到对应的视觉证据支撑，是多模态事实一致性的核心要求。

## 核心要点
1. 与[[语言先验]]对立：视觉接地强的 token 依赖完整视觉通路，语言先验强的 token 即使移除视觉也可保持高分
2. 多模态 Transformer 的早期层负责将视觉信息注入文本token表示，形成视觉接地
3. 是[[幻觉抑制]]的核心目标：保留视觉接地强的 token，抑制语言先验主导的 token
4. 属性级微调错误（如光照导致的颜色误判）是视觉接地的边缘案例

## 代表工作
- [[SIRA]]: 通过全分支与反事实分支的 logit 差异近似测量每个 token 的视觉接地程度
