# 多模态幻觉

## 定义
多模态大模型在生成回答时，输出内容与感知输入（图像/音频/视频）不一致或包含感知信号中不存在的信息的现象。

## 核心要点
1. **成因**: 模型过度依赖语言先验（文本 token 主导），感知模态 token 未被充分利用
2. **表现**: 描述不存在的对象、错误属性、虚构关系、错误的事件时间定位
3. **评测**: [[POPE]]（二分类对象存在性）、[[CHAIR]]（描述级幻觉比例）、AMBER、MME 等
4. **缓解**: 对比解码（[[VCD]]/[[ICD]]）、推理时干预（[[LIME]]/[[V-ITI]]）、RLHF 对齐、训练数据增强
5. 是当前多模态大模型部署的核心挑战之一，尤其在医疗、自动驾驶等高可靠性场景

## 代表工作
- [[LIME]]: 基于 LRP 的推理时幻觉缓解框架
- [[VCD]]: Visual Contrastive Decoding
- [[OPERA]]: 基于 attention map 的幻觉缓解
